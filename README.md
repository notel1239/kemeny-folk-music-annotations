# Kemeny Folk Music Annotations

## Project Overview

This repository serves as a digital humanities module for a course on Hungarian cultural history. The focus is on extracting and contextualizing musical references from primary sources, specifically a digitized excerpt from the autobiography of Prince John Kemeny of Transylvania.

## Workflow

The project demonstrates a complete workflow for collaborative annotation using GitHub features:

1. **Source Text**: The file `kemeny_excerpt.md` contains an excerpt from Kemeny's autobiography.
2. **Annotation Identification**: GitHub Issues are used to identify terms requiring annotation.
3. **Annotation Proposal**: Pull Requests are used to propose and discuss annotations.
4. **Review Process**: Pull request reviews and inline comments facilitate collaborative refinement.
5. **Finalization**: Approved annotations are merged into the main branch.

## Getting Started

Students should:
1. Fork this repository
2. Create a new branch for annotations
3. Edit `kemeny_excerpt.md` with proposed annotations using Markdown blockquotes
4. Open a Pull Request with a descriptive title
5. Participate in the review process
